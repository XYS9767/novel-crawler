# 📚 番茄小说爬虫 (Novel Crawler)

一个功能强大的番茄小说爬取工具，支持搜索下载、字体解码、GUI 界面等功能。

## ✨ 功能特性

- 🔍 **智能搜索**：支持通过小说名称或作者搜索
- 🔗 **URL 直链**：支持直接使用小说 URL 爬取
- 📖 **章节选择**：可自定义爬取章节范围（从第 X 章到第 Y 章）
- 🔤 **字体解码**：自动解码番茄小说的加密字体
- 🎨 **图形界面**：基于 tkinter 的友好 GUI 界面
- 📊 **进度显示**：实时显示爬取进度和统计信息
- 🛡️ **反爬对抗**：
  - User-Agent 轮换
  - 智能延迟机制
  - 会话保持
  - 重试机制（支持 403、404、503 等错误处理）
- 📁 **批量下载**：支持批量下载多个章节

## 🚀 快速开始

### 方式一：直接使用可执行文件（推荐）

双击运行 `番茄小说爬虫.exe` 即可启动图形界面。

### 方式二：Python 运行

#### 1. 环境要求

- Python 3.8+
- Windows 10/11

#### 2. 安装依赖

```bash
cd python-backend
pip install -r requirements.txt
```

#### 3. 启动 GUI 界面

```bash
python run_gui.py
```

## 📖 使用说明

### GUI 界面使用

1. **启动程序**：运行 `番茄小说爬虫.exe` 或 `python run_gui.py`
2. **输入小说信息**（二选一）：
   - **小说名称**：输入要爬取的小说名称（如：`完美世界`）
   - **小说 URL**：直接粘贴小说页面 URL（优先级更高）
     - 格式：`https://fanqienovel.com/page/xxxxxx`
     - 或：`https://fanqienovel.com/reader/xxxxxx`
3. **可选配置**：
   - **作者名称**：用于过滤搜索结果
   - **最小章节数**：从第几章开始爬取（默认：1）
   - **最大章节数**：最多爬取多少章（默认：100）
   - **保存路径**：小说保存的目录
4. **开始爬取**：点击"▶ 开始爬取"按钮
5. **查看进度**：
   - 进度条显示整体进度
   - 日志窗口显示详细信息
   - 统计信息显示已找到和已下载的小说数量

### 命令行使用（高级）

```python
from crawler.fanqie_crawler import FanqieCrawler
from utils.config import Config

# 创建配置
config = Config(
    novel_name="完美世界",
    author="辰东",
    save_path="./novels/",
    min_chapters=1,      # 从第 1 章开始
    max_chapters=50,     # 爬到第 50 章
)

# 创建爬虫
crawler = FanqieCrawler(config)

# 开始爬取
crawler.start()
```

## 🔧 配置说明

### Config 类参数

| 参数                | 类型  | 默认值         | 说明            |
| ----------------- | --- | ----------- | ------------- |
| `novel_name`      | str | ""          | 小说名称          |
| `novel_url`       | str | ""          | 小说页面 URL（可选）  |
| `author`          | str | None        | 作者名称（可选）      |
| `save_path`       | str | "./novels/" | 保存路径          |
| `min_chapters`    | int | 1           | 最小章节数（从第几章开始） |
| `max_chapters`    | int | 100         | 最大章节数         |
| `max_workers`     | int | 3           | 最大工作线程数       |
| `request_timeout` | int | 10          | 请求超时时间（秒）     |

## 📝 技术细节

### 字体解码

番茄小说使用自定义字体文件加密内容，本项目通过字符映射表自动解码：

- 编码范围：`U+E5E8` - `U+E753`
- 映射表：`CHARSET`（包含 371 个字符）
- 解码器：`font_decoder.py`

### 反爬策略

1. **User-Agent 轮换**：每 20 章更换一次 User-Agent
2. **智能延迟**：
   - 首次请求：1-2 秒
   - 重试请求：2-5 秒递增
3. **会话保持**：使用 `requests.Session` 保持连接
4. **错误重试**：支持最多 5 次重试

### 章节获取方式

1. **API 解析**：从 `__INITIAL_STATE__` 提取章节列表
2. **HTML 解析**：从页面元素提取章节信息
3. **连续爬取**：通过"下一章"链接连续获取

## 🐛 常见问题

### 1. 爬取失败/403 错误

**原因**：触发反爬机制

**解决方法**：

- 等待一段时间后重试
- 减少单次爬取章节数
- 检查网络连接

### 2. 章节内容为乱码

**原因**：字体解码失败

**解决方法**：

- 确保 `font_decoder.py` 文件存在
- 检查字符映射表是否完整

### 3. 找不到章节列表

**原因**：页面结构变化

**解决方法**：

- 程序会自动切换到连续爬取模式
- 确保 URL 格式正确

### 4. GUI 界面无法启动

**原因**：缺少 tkinter 依赖

**解决方法**：

```bash
# Windows 通常自带 tkinter
# 如缺失，重新安装 Python 并勾选 tcl/tk 支持
```
